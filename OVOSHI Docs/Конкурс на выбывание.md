В конкурсах участвуют все игроки, конкурс может быть отменен на усмотрение Лидера и Заместителей. Конкурсы на выбывание проводятся если в клане более 45 человек.
## Еженедельный конкурс выбывания
Кикаются участники с 0 недельной энергии, если нет с 0, значит последний по количеству.
## Ежесезонный конкурс выбывания
Кикаются участники с 0 сезонной энергии, если нет с 0, значит последний по количеству.
## Выбывание по битве клана
Кикаются все у кого ноль очков по окончанию битвы, если нет с 0, значит последний по количеству.
